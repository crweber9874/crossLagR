4  The Random-Intercept CLPM

Chapter 1 showed that the CLPM conflates stable between-person differences with within-person dynamics. Chapter 2 addressed this through change scores; the Latent Change Score Model. This chapter takes up the alternative decomposition popularized by Hamaker, Kuiper, and Grasman (2015): the Random-Intercept Cross-Lagged Panel Model (RI-CLPM), which splits each observed score into a stable trait (\(I_x\), \(I_y\)) and a wave-specific within-person deviation (\(p_t\), \(q_t\)). Lagged regressions are then specified on the deviations, not on the raw scores.

4.1 Decomposition

\(x_{it}\) and \(y_{it}\) denote the observed scores for person \(i\) at time \(t\). It’s useful to decompose the observed scores into constituent parts: an overall mean for each wave (\(\mu_{xt}, \mu_{yt}\)), a stable trait for each person (\(I_{x,i}, I_{y,i}\)), and a time-varying deviation from that trait (\(x^*_{it}, y^*_{it}\)). Or more compactly, the effect of time, the effect of person, and the effect of random noise around these two components. \[ \begin{aligned} x_{it} &= \mu_{xt} + I_{x,i} + x^*_{it}, \\ y_{it} &= \mu_{yt} + I_{y,i} + y^*_{it}, \end{aligned} \]

Unlike the CLPM though, the dynamic equation is not specified on the observed scores, but rather, the deviations from the trait, \(x^*_{it}\) and \(y^*_{it}\). \[ \begin{aligned} x^*_{it} &= \text{ar}_x\,x^*_{i,t-1} + \text{cl}_{yx}\,y^*_{i,t-1} + u_{x,it}, \\ y^*_{it} &= \text{ar}_y\,y^*_{i,t-1} + \text{cl}_{xy}\,x^*_{i,t-1} + u_{y,it}. \end{aligned} \]

NotecrossLagR syntax: estimateRICLPM()

estimateRICLPM() emits lavaan syntax with the unified labels ar_x, ar_y (within-person carry-over), cl_xy, cl_yx (within-person cross-lags), I_x, I_y (stable traits), and d_var_*, d_cov_xy (dynamic residuals). Optional include_z = TRUE adds a third process; time_invariant_vars regresses traits on covariates.

estimateRICLPM(waves           = 3,
               constrain_beta  = TRUE,
               constrain_omega = TRUE) |>
  cat()

# Random Intercepts (Trait Factors I)
I_x =~ 1*x1 + 1*x2 + 1*x3
I_y =~ 1*y1 + 1*y2 + 1*y3

# Within-Person Latent Variables
p1 =~ 1*x1
q1 =~ 1*y1
p2 =~ 1*x2
q2 =~ 1*y2
p3 =~ 1*x3
q3 =~ 1*y3

# Within-person factor means (fixed to 0)
p1 ~ 0*1
q1 ~ 0*1
p2 ~ 0*1
q2 ~ 0*1
p3 ~ 0*1
q3 ~ 0*1

# Trait factor means (free)
I_x ~ 1
I_y ~ 1

# Observed intercepts (fixed to 0)
x1 ~ 0*1
y1 ~ 0*1
x2 ~ 0*1
y2 ~ 0*1
x3 ~ 0*1
y3 ~ 0*1

# Autoregressive and Cross-lagged Effects
p2 ~ ar_x*p1 + cl_yx*q1
q2 ~ ar_y*q1 + cl_xy*p1
p3 ~ ar_x*p2 + cl_yx*q2
q3 ~ ar_y*q2 + cl_xy*p2

# Dynamic Residual Variances
p1 ~~ d_var_x*p1
q1 ~~ d_var_y*q1
p2 ~~ d_var_x*p2
q2 ~~ d_var_y*q2
p3 ~~ d_var_x*p3
q3 ~~ d_var_y*q3

# Dynamic Residual Covariances
p1 ~~ d_cov_xy*q1
p2 ~~ d_cov_xy*q2
p3 ~~ d_cov_xy*q3

# Trait Factor (I) Variances and Covariances
I_x ~~ I_x
I_y ~~ I_y
I_x ~~ I_y

# Fix Observed Variable Residuals to Zero
x1 ~~ 0*x1
y1 ~~ 0*y1
x2 ~~ 0*x2
y2 ~~ 0*y2
x3 ~~ 0*x3
y3 ~~ 0*y3
NotecrossLagR syntax: simRICLPM()

simRICLPM() simulates from the RI-CLPM at user-specified true values. var_BX, var_BY set the trait variances, var_p, var_q the within-person dynamic-residual variances, and beta_x/beta_y, omega_xy/omega_yx the AR and CL parameters.

sim <- simRICLPM(waves       = 5,
                 beta_x      = 0.3,
                 beta_y      = 0.3,
                 omega_xy    = 0.2,
                 omega_yx    = 0.1,
                 var_BX      = 1.0,
                 var_BY      = 1.0,
                 var_p       = 1.0,
                 var_q       = 1.0,
                 sample.nobs = 1000)
